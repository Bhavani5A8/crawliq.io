Async Technical SEO Crawler

Crawl up to 500 pages using a breadth-first async crawler built with Python aiohttp and BeautifulSoup. The crawler fetches pages concurrently — a 100-page site typically completes in under 60 seconds. Each page is scored on over 50 on-page and technical signals including title tags, meta descriptions, heading structure, canonical correctness, canonical loops, internal link counts, image alt coverage, Open Graph completeness, and page-level crawlability. Multi-Model AI Fix Engine

Route your crawl report to any of four AI providers: Groq (Llama 3.3 — fastest), Google Gemini 1.5, Anthropic Claude 3 Haiku, or OpenAI GPT-4o. Technical Capabilities

What the Crawler Detects

CrawlIQ's crawler checks every page for over 50 SEO signals including:

  • Missing or duplicate title tags; titles outside 30–60 character range
  • Missing or duplicate meta descriptions
  • Missing H1, multiple H1s, or empty heading structure
  • Canonical tag absent, self-referencing, or pointing to a different URL
  • Pages marked noindex or nofollow unintentionally
  • Broken internal links (4xx and 5xx HTTP status codes)
  • Redirect chains (3xx → 3xx) wasting crawl budget
  • Images missing alt text attributes
  • Thin content pages with under 300 words
  • Open Graph and Twitter Card tag coverage
  • Core Web Vitals proxies (LCP, CLS, FID indicators)
  • HTTPS enforcement and mixed-content warnings

Excel Export Formats

Every audit can be exported to a structured Excel workbook with multiple sheets:

  • Full SEO Report — all crawled pages with scores, issues, and keywords
  • Live Optimisation Table — paste-ready optimised titles, descriptions, and H1s
  • Technical SEO Audit — per-page scores (0–100) with A–F grades
  • Per-Field Issues — field-level breakdown for bulk fixing workflows
  • AI Generated Content — full AI output per page for content teams

AI Provider Comparison

  • Groq + Llama 3 — fastest response time; free API at console.groq.com
  • Google Gemini — strong multilingual and long-context performance
  • Anthropic Claude — best for nuanced content quality assessment
  • OpenAI GPT-4o — industry-standard benchmark model
  • Rules-based fallback — works with zero API key; deterministic issue detection
