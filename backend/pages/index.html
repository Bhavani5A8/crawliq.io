Got questions?

Frequently Asked

Is CrawlIQ really free?
▼
Yes. Guest users can crawl up to 50 pages per session. Free accounts (no credit card needed) get 200 pages/month across 3 projects. Pro and Agency plans unlock higher limits, team seats, and white-label PDF exports.
Which AI provider should I use?
▼
Groq (Llama 3) is free with a generous API quota and the fastest. Gemini Flash is free with a Google API key. OpenAI and Claude are paid but offer the most nuanced fixes. Ollama runs locally with no API costs at all.
Can I crawl competitor sites?
▼
Yes — the competitor analysis module lets you audit up to 5 URLs in one run, compare dimension scores, identify keyword gaps, and get an AI-ranked action plan. You can crawl any publicly accessible site.
Does it work on JavaScript-rendered sites?
▼
CrawlIQ uses curl_cffi to bypass Cloudflare, DataDome, PerimeterX and other bot protection systems. For JS-heavy sites, it still crawls all linked HTML pages — it does not execute JavaScript like a headless browser.
Is my data private?
▼
Crawl results are stored per-session in memory and optionally in your project history if you're logged in. We never sell data to third parties. API keys you enter are stored server-side per user and never exposed in logs.
