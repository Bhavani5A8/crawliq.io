HOW IT WORKS

Four Steps from Crawl to Ranked

CrawlIQ runs a full SEO pipeline automatically — no configuration needed

1

Enter Your URL

Paste any website URL. CrawlIQ starts a breadth-first crawl, discovers all internal links, handles SSL and HTTPS fallback automatically, and processes up to 50 pages per session at no cost.

2

Technical SEO Audit

Every crawled page is automatically scored 0–100 across 9 technical dimensions: title tag, meta description, canonical, heading structure, Open Graph, content depth, URL quality, image alt text, and HTTP status.

3

AI Fix Generation

Run AI analysis on any page using Groq Llama 3, Gemini, OpenAI GPT, or Claude. The AI generates optimized titles, meta descriptions, H1 tags, and full content rewrites — ranked by impact, ready to paste.

4

Export & Implement

Download a full Excel (.xlsx) audit report with per-field issues, AI fixes, keyword scores, ranking grades (A–F), and a Live Optimization Table containing all paste-ready improvements for your dev or content team.

FAQ

Frequently Asked Questions

Everything you need to know about CrawlIQ

How does CrawlIQ's AI SEO analysis work?
▲
CrawlIQ crawls your website using an async BFS crawler and sends each page to an AI model — Groq (Llama 3), Gemini, OpenAI, or Claude (configurable). The AI analyzes content quality, keyword optimization, readability, and semantic relevance, then generates specific, actionable recommendations ranked by impact. Output includes optimized title tags, meta descriptions, H1 rewrites, and full paragraph content.
What SEO issues does CrawlIQ detect?
▼
CrawlIQ detects 15+ issue categories: missing or oversized title tags, missing meta descriptions, missing or multiple H1 tags, no H2 structure, canonical tag issues, duplicate meta descriptions across pages, broken pages (4xx/5xx errors), thin content (under 300 words), images missing alt text, HTTP instead of HTTPS, and URL problems like underscores or excessive depth.
What data formats can I export?
▼
CrawlIQ exports four Excel (.xlsx) reports: a Full SEO Report with all pages, issues, and AI fixes; a Per-Field Issues Report; a Live Optimization Table with paste-ready optimized content; and a Technical SEO Audit with per-page scores (0–100), letter grades (A–F), indexability status, and component-level breakdowns.
How many pages can I crawl?
▼
CrawlIQ is free with no credit card required. As a guest you can crawl up to 50 pages per session — enough for most small to medium sites. Creating a free account raises this to 200 pages per month with project saving and rank monitoring. Pro and Agency plans are available for larger sites. The crawler uses breadth-first search from your root URL, follows internal links, and handles SSL failures with automatic HTTP fallback.
Is my website data secure and private?
▼
Yes. All crawl data is processed in isolated in-memory sessions and never stored permanently or shared. Data exists only for the duration of your session and is cleared when a new crawl starts. CrawlIQ is fully open-source on GitHub and hosted on HuggingFace Spaces — you can inspect every line of code.
Does CrawlIQ support JavaScript-rendered sites?
▼
CrawlIQ uses an aiohttp-based async crawler with BeautifulSoup for HTML parsing. It works best with server-side rendered, static, and WordPress sites, extracting titles, meta tags, headings, canonicals, Open Graph tags, body text, and image alt texts. For heavily client-side SPAs, enabling server-side rendering gives the best crawl coverage.
