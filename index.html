FAQ

Frequently Asked Questions

Common questions about the CrawlIQ technical SEO evaluation reference

psychology How does CrawlIQ's AI SEO evaluation work? expand_more
CrawlIQ examines each page of a website using an async BFS evaluation engine and sends each page's signal data to an AI model — Groq (Llama 3), Gemini, OpenAI, or Claude (configurable). The AI evaluates content quality, keyword alignment, readability, and semantic relevance, then produces structured remediation recommendations ranked by impact. Output includes optimised title tags, meta descriptions, H1 rewrites, and content guidance.
bug_report What SEO issues does CrawlIQ identify? expand_more
CrawlIQ identifies 15+ issue categories: missing or oversized title tags, missing meta descriptions, missing or multiple H1 tags, no H2 structure, canonical tag issues, duplicate meta descriptions across pages, pages returning 4xx/5xx errors, thin content (under 300 words), images missing alt text, HTTP instead of HTTPS, and URL structure issues like underscores or excessive depth.
download What data formats can I export? expand_more
CrawlIQ exports structured Excel (.xlsx) reference documents: a Full SEO Report with all evaluated pages, issues, and AI remediation guidance; a Per-Field Issues Report; a Remediation Table with optimised content; and a Technical SEO Assessment with per-page scores (0–100), letter grades (A–F), and indexability status.
travel_explore How many pages can the evaluation engine process? expand_more
CrawlIQ is free with no credit card required. As a guest you can evaluate up to 50 pages per session — enough for most small to medium sites. Creating a free account raises this to 200 pages per month with project saving and position tracking. The evaluation engine uses breadth-first search from your root URL and handles SSL failures with automatic HTTP fallback.
lock Is my website data secure and private? expand_more
Yes. All evaluation data is processed in isolated in-memory sessions and never stored permanently or shared. Data exists only for the duration of your session and is cleared when a new evaluation begins. CrawlIQ is fully open-source on GitHub and hosted on HuggingFace Spaces — you can review every line of code.
javascript Does CrawlIQ support JavaScript-rendered sites? expand_more
CrawlIQ uses an aiohttp-based async evaluation engine with BeautifulSoup for HTML parsing. It works best with server-side rendered, static, and WordPress sites, extracting titles, meta tags, headings, canonicals, Open Graph tags, body text, and image alt texts. For heavily client-side SPAs, enabling server-side rendering gives the best evaluation coverage.
